# jpgmagick
A qt tool for changing the contrast and brightness of jpg images with ImageMagick.

Syntax:
```
  jpgmagick [directory]
```

Example:
```
  jpgmagick ~/photos
```

Requirements:
  * [ImageMagick (convert)](https://www.imagemagick.org)
